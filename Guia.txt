Ejecuta el código Python para iniciar el juego de bingo.
El programa generará automáticamente un cartón de bingo con 5 números aleatorios entre 1 y 20 para el usuario.
El programa también generará 5 números aleatorios para el bingo.
El objetivo es acertar la mayor cantidad de números posibles entre los generados para el usuario y los números del bingo.
El programa mostrará cuántos números acertaste.
Los premios se asignan según la cantidad de aciertos:
5 aciertos: ¡Has ganado un millón de dólares!
4 aciertos: ¡Has ganado 500 mil dólares!
3 aciertos: ¡Has ganado otro cartón!
Si tienes menos de 3 aciertos, se te animará a seguir intentando.
¡Diviértete jugando al bingo!
